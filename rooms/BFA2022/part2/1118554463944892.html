Infiltrating the Network

 With a successful login to the network, your team buckles down to go digital. You can see all non-admin nodes of the network by using a custom ping command on your terminal; details like the node name and transmission function appear instantly. Specifically, the transmission function collects data about traffic going through the node before sending it where it needs to be.

 The first node that you access seems suspicious. The source code for the transmission function changes the ORIGINAL data, rather than monitoring the data. This should never happen for security reasons - the hackers changed the transmission functions so that they could impersonate as nodes on the network. Finding the names of all the nodes that modify original data will allow you to build a map to track the attackers.

Objective: Collect the names of breached nodes.
