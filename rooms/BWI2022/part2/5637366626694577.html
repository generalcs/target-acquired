Hijack the Cows' Quarters

 The escape path has been set. Now it’s time to release the cows from their iron cages. You exit your spacecraft cautiously and peer through the window to get a better look. The cages of the cows are controlled by a common gate lock in the middle of the quarters. Thankfully, the farmer did not encrypt the code to the lock securely; he encoded it as a postorder traversal of a tree.

 In your notepad below, you’ve written down an array representation of the tree. The elements start at index one i.e. the root node is at index one. If you can load this representation onto your computer and perform a postorder, you should be able to break through the window and quickly release the gates. Time is ticking - you never know when the next farm patrol will arrive.

Objective: Decode the encrypted gate code tree.
